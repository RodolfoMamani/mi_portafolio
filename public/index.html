Proyectos implementados

Desarrollo de Sistema Web

Frontend desarrollado con el framework de JavaScript Vue.js, utilizando componentes UI de Vuetify. Backend implementado con PocketBase (BaaS), utilizando una base de datos no relacional accesible desde cualquier frontend.

Implementación de Servicio VoIP

Se implementó un servicio VoIP que permite realizar llamadas en una red local por medio del protocolo IP. Se utilizó la herramienta Asterisk sobre un sistema operativo Ubuntu­Linux. Aplicación de conceptos de Protocolos de Comunicación (INF 319).

Diseño e Implementación de Base de Datos

Se realizó el diseño conceptual con diagramas entidad­relación (23 entidades) y normalizado hasta la tercera forma normal. Base de datos implementada en Oracle SGBD, incluyendo funciones y procedimientos almacenados. Proyecto realizado en el Taller de Base de Datos (INF 272).

Sistema de Administración de Movilidad

Proyecto enfocado en el registro y organización de vehículos y sus operadores. Desarrollado con NetBeans y Java, aplicando conceptos de estructuras de datos, Programación Orientada a Objetos (POO) y UML. Proyecto de Estructura de Datos (LAB 131).
